Relatório Parte 1
Em cria_pacientes.sh utilizo o awk para separar as linhas sempre que aparece “:” ou uma “,”.  Vou à procura das linhas que começam com “a” e a seguir têm pelo menos um número, no ficheiro /etc/passwd, para me certificar que estou a tratar apenas os alunos.  Depois separei o a do resto do número de aluno.  Sendo assim tenho todos os dados que preciso para criar uma lista de pacientes, logo imprimo os dados pela ordem pedida separados por “;” e envio as primeiras dez linhas para o ficheiro “pacientes.txt”.

Em cria_medico.sh, começo por verificar que o utilizador coloca quatro argumentos. Depois verifico se o ficheiro “medicos.txt” já existe. Caso já exista vou ver se o número do médico que se pretende criar é diferente dos números dos médicos já existentes. Caso seja diferente, adiciono o novo medico ao ficheiro, caso não seja diferente não o coloco no ficheiro. Se o ficheiro “medicos.txt” ainda não existir imprimo logo o médico que se pretende criar num novo ficheiro “medicos.txt”. No final do script, independentemente de se o novo médico foi adicionado ou não, imprimo o conteúdo do ficheiro “medicos.txt”.

Em stats.sh, começo por verificar que o utilizador fornece os dois argumentos necessários para o script. Para o número de pacientes de uma determinada localidade, vou contar o número de linhas que têm a localidade dada na sua sintaxe. Para o número de médicos que têm um saldo superior ao saldo colocado, utilizo o awk para separar os dados dos médicos verifico se o saldo de cada medico é superior ao saldo colocado pelo utilizador, se essa condição for verdade imprimo o saldo. No final conto o número de linhas impressas que é igual o número de médicos com o saldo superior.

Em avalia_medicos.sh, utilizo o awk para verificar se o número de consultas é superior a 6 e o ranking é inferior a 5. Se essa condição se verificar imprimo o médico. No final envio a lista de médicos para um novo ficheiro “lista_negra_medicos.txt” e imprimo o conteúdo desse ficheiro para o utilizador.

Em menu.sh, início uma variável “option” como um número diferente dos que aparecem no menu. Entre em um ciclo while que se vai repetir até o utilizador selecionar “0”. Utilizando um case consigo que o programa corra o script selecionado pelo utilizador. No caso da opção “2” e “3” vai pedindo ao utilizador e lendo os argumentos necessários.
